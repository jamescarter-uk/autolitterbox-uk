## About the Editor

Auto Litter Box UK is written and edited by **James Carter**, a tech-focused pet care researcher based in the UK.

James tests automatic litter boxes in real-world conditions with his own cats in a small Manchester flat, focusing on what matters most to UK cat owners:

- reliability  
- noise at night  
- odour control in smaller homes  
- total long-term cost of ownership  

All reviews are written for UK readers, using UK pricing and availability wherever possible.
